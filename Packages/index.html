RiscPkgs

These are most of the programs from the rest of the site experimentally packaged up into RiscPkgs.

To use them with !Riscpkg, edit <Choices$Write>.RiscPkg.Sources to include the following line of text:

pkg http://resurgam-riscos.github.io/Packages/Unstable.txt

To use the EzROOL package you must ensure that your Sources file also contains the following line: (Because it uses the Bzip2 package)

pkg http://www.riscos.info/packages/pkg/autobuilt

(if you don't have a <Choices$Write>.RiscPkg.Sources file then just copy !RiscPkg.Resources.Sources to <Choices$Write>.RiscPkg.Sources)

Then run !RiscPkg, open the main window and choose 'Update' from the main menu. All the Resurgam packages should appear in the window, you can then install any you wish.

For further information please refer to the RiscPkg manual.
